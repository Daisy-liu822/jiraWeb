# 🚀 快速启动指南

## Windows 一键启动 ArgoCD 工具

本地运行避免网络限制，ArgoCD 功能完全可用。

### 方法 1: 批处理文件（推荐）⭐

**双击运行：**
```
启动ArgoCD工具.bat
```

**特点：**
- ✅ 双击即可运行，无需其他操作
- ✅ 自动检查并安装依赖
- ✅ 自动打开浏览器到 ArgoCD 页面
- ✅ 支持虚拟环境自动激活

---

### 方法 2: PowerShell 脚本

**右键 → "使用 PowerShell 运行"：**
```
启动ArgoCD工具.ps1
```

**如果遇到执行策略限制：**
```powershell
# 以管理员身份运行 PowerShell，执行：
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

# 或者直接运行：
powershell -ExecutionPolicy Bypass -File "启动ArgoCD工具.ps1"
```

**特点：**
- ✅ 彩色输出，界面更友好
- ✅ 更详细的错误处理
- ✅ 功能与批处理相同

---

# 1. 打开 PowerShell 或 CMD

## 📋 首次使用步骤

### 1. 确保已安装 Python

下载：https://www.python.org/downloads/

**检查安装：**
```powershell
python --version
```

应该显示：`Python 3.9.x` 或更高版本

### 2. 安装依赖（首次运行）

```powershell
pip install -r requirements.txt
```

### 3. 运行启动脚本

双击 `启动ArgoCD工具.bat`

---

## 🎯 使用说明

### 启动后

1. **浏览器自动打开** → `http://localhost:8501`
2. **主页** → 点击 "打开 ArgoCD 工具"
3. **或直接访问** → `http://localhost:8501/ArgoCD_Images`

### ArgoCD 工具使用

1. **选择环境**：preprod / staging / prod
2. **输入 Token**：从 ArgoCD UI Settings → Tokens 获取
3. **选择服务**：从列表中选择要查询的服务
4. **开始查询**：点击按钮查看结果
5. **查看对比**：再次查询时自动显示变化高亮

## 🔧 故障排除

### 问题 1: Python 未找到

**解决：**
- 安装 Python：https://www.python.org/downloads/
- 安装时勾选 "Add Python to PATH"

### 问题 2: 依赖安装失败

**解决：**
```powershell
# 升级 pip
python -m pip install --upgrade pip

# 重新安装依赖
pip install -r requirements.txt
```

### 问题 3: 端口被占用

**解决：**
```powershell
# 使用其他端口
streamlit run app.py --server.port 8502
```

### 问题 4: PowerShell 脚本无法运行

**解决：**
```powershell
# 方法1：修改执行策略
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

# 方法2：使用批处理文件
# 双击 启动ArgoCD工具.bat
```

## 🚀 快捷方式（可选）

### 创建桌面快捷方式

1. 右键 `启动ArgoCD工具.bat`
2. 发送到 → 桌面快捷方式
3. 重命名为 "DevOps 工具集"
4. 可选：右键快捷方式 → 属性 → 更改图标

这样就可以从桌面一键启动！
